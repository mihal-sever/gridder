# Gridder
App helps artists to mark up a canvas with checkpoints to ease digital sketch transfer.

# How to Use

Press **new project** and select an image.

<img width="1440" alt="Screenshot 2022-09-07 at 17 42 42" src="https://user-images.githubusercontent.com/49707539/188894527-453ce2d1-d756-4c8f-b56f-5982dfbe33e2.png">

Set project name, canvas width or height and grid step in millimeters, then press **apply**.

<img width="1440" alt="Screenshot 2022-09-07 at 17 43 18" src="https://user-images.githubusercontent.com/49707539/188894688-a69c22be-b218-4607-b665-ec5b4bec547b.png">

Click on spare space to create a reference point of the chosen colour.

<img width="1440" alt="Screenshot 2022-09-07 at 17 44 44" src="https://user-images.githubusercontent.com/49707539/188895056-3fd3d6b0-45c9-4957-9d68-4c001cbee6f8.png">

Click the point to see its coordinates to closest leftmost topmost grid node.

<img width="1433" alt="Screenshot 2022-09-07 at 17 44 51" src="https://user-images.githubusercontent.com/49707539/188895306-08da59ee-3ca8-40e3-a3c5-5929b2e68b58.png">

After transfering the point to the canvas press **ctrl+f** to mark it as finished - point will change its colour to red. Press **ctrl+f** to mark it as unfinished.

<img width="1436" alt="Screenshot 2022-09-07 at 17 45 57" src="https://user-images.githubusercontent.com/49707539/188896642-985d60a0-c1f1-4943-a7e5-5f41a473083b.png">


## Hotkeys

- **ctrl/cmd+s** - save project
- **ctrl/cmd+d** - delete selected point
- **ctrl/cmd+f** - mark point as finished/unfinished
- **ctrl/cmd+z** - undo last command
